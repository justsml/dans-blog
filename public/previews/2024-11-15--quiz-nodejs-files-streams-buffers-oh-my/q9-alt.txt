# 9. File System

How many times is fs.watch() guaranteed to fire when a file is modified?

import fs from 'fs';
fs.watch('test.txt', (eventType, filename) => {
  console.log(`${filename} was changed`);
});
// Then modify test.txt once
------

1. Once per file change
2. Not guaranteed; may fire multiple times
3. Never
4. Only on file deletion
