# 8. Conditionals

How can we compare strings in Bash?

Terminal window
cat1="Rosie"
cat2="Sunflower"
if [ "$cat1" === "$cat2" ]; then
  echo "Same cat"
else
  echo "Different cats"
fi
------

1. Same cat
2. Different cats, after a test syntax error
3. Zalgo
4. Error only
