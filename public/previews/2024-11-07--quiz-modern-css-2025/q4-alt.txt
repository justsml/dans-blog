# 4. CSS Grid Functions

What is the effect of using minmax(100px, 200px) for a CSS grid track?

grid-template-columns: minmax(100px, 200px);
------

1. All column widths between 100px & 200px
2. Set Columns to 100px, Rows to 200px
3. Column one will be between 100px & 200px
4. Apply range recursively, incl. subgrids
