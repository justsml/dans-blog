# 8. CSS Nesting

Is this a correct use of native CSS nesting?

.container {
  color: black;
  .title {
    color: white;
    background: black;
  }
}
------

1. Filename must end w/ .scss
2. `.title` must precede properties like `color`
3. Only w/ PostCSS
4. Perfect. No notes.
