# 3. SQL Joins

What does a LEFT JOIN do in SQL?
------

1. Returns one row per every matching row, ignores unmatched rows
2. Returns all rows from the left table, plus matching right rows
3. Returns all matched or unmatched rows, where NULLs indicate no match
4. Returns all rows from the right table, with any matching left rows
