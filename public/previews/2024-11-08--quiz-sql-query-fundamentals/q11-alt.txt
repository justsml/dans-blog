# 11. SQL Joins

What does a FULL OUTER JOIN do in SQL?
------

1. Returns only matched rows
2. Returns only rows with non-NULL values
3. Returns unmatched rows from the left table only
4. Returns unmatched rows from the right table only
5. Returns all rows from both tables, with NULLs for unmatched rows
