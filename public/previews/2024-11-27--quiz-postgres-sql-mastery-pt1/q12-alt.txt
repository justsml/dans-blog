# 12. Date/Time

Which is the smallest timestamp literal to exceed the maximum time precision in Postgres?

CREATE TABLE class_sessions (
  id INT GENERATED BY DEFAULT AS IDENTITY,
  start_time timestamptz,
  end_time timestamptz
);
------

1. 2024-01-08 13:30:00+00
2. 2024-01-08 13:30:00.123456+00
3. 2024-01-08 13:30:00.123456789+00
4. 2024-01-08 13:30:00.1234567
