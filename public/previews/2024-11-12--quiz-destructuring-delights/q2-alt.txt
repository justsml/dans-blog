# 2. Warmup: Arrays

What will this code do?

const person = [ 'Dan Levy', 'Cape Town' ];
const [ name, origin, age ] = person;
console.log(`Name: ${name}, Age: ${age}`);
------

1. Name: Dan Levy, Age: NaN
2. Name: Dan Levy, Age: null
3. Name: Dan Levy, Age: undefined
4. Name: Dan Levy, Age: 40
5. Error: Cannot destructure property 'age'
6. SyntaxError: Unexpected token ','
