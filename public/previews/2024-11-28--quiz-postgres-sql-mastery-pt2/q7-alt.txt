# 7. Quoting 1

Which is correct for case-sensitive column names?

CREATE TABLE grades (
  "Student Name" VARCHAR(100),
  'Student Grade' INTEGER
);
------

1. Both work the same
2. Double quotes are required
3. Single quotes are required
4. Syntax error: Invalid user literal
5. All names MUST be lowercase
