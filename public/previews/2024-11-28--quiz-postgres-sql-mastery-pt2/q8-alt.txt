# 8. Quoting 2

How do you specify a column name containing a period?

CREATE TABLE teachers (
  id INT GENERATED BY DEFAULT AS IDENTITY,
  first.name TEXT,
  -- How do you include the column: first.name?
  salary NUMERIC
);
------

1. first.name
2. "first.name"
3. 'first.name'
4. [first.name]
5. [first].[name]
