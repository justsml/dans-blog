  </div>
  </slot>

  <slot name='explanation'>
  <div className="explanation">
    تُستخدم عبارة `WITH`، أو التعبير الجدولي المشترك (CTE)، لتعريف مجموعة نتائج مؤقتة يمكن الرجوع إليها داخل الاستعلام الرئيسي. في PostgreSQL، يمكن إرفاق `WITH` بعبارات مثل `SELECT`، `INSERT`، `UPDATE`، `DELETE`، أو `MERGE`.

    يساعد ذلك في تحسين قابلية القراءة والصيانة للاستعلامات المعقدة.

    الصيغة هي:
    ```sql
        WITH cte_name AS (
        SELECT column_name
        FROM table_name
        )
        SELECT *
        FROM cte_name;
    ```
  </div>
  </slot>
