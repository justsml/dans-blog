  <slot name="question">
  <div className="question">
    ¿Para qué sirve realmente la cláusula `WITH` (Expresión de Tabla Común) en SQL?
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    La cláusula `WITH`, o Expresión de Tabla Común (CTE), se utiliza para definir un conjunto de resultados temporal que puede referenciarse dentro de la consulta principal. En PostgreSQL, `WITH` puede adjuntarse a sentencias como `SELECT`, `INSERT`, `UPDATE`, `DELETE` o `MERGE`.
    ```sql
    WITH cte_name AS (
    SELECT column_name
    FROM table_name
    )
    SELECT *
    FROM cte_name;
    ```
  </div>
  </slot>
