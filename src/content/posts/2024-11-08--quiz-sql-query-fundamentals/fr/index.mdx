    {text: 'Utilisé pour les fonctions d\'agrégation'},
    {text: 'Utilisé uniquement pour définir des vues'},
    {text: 'Définit un ensemble de résultats temporaire pour une utilisation dans une instruction SQL plus large', isAnswer: true},
  ]}
>
  <slot name="question">
  <div className="question">
    Quel est le but de la clause `WITH` (Common Table Expression) en SQL ?
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    La clause `WITH`, ou Common Table Expression (CTE), est utilisée pour définir un ensemble de résultats temporaire qui peut être référencé dans la requête principale. Dans PostgreSQL, `WITH` peut être attaché à des instructions telles que `SELECT`, `INSERT`, `UPDATE`, `DELETE` ou `MERGE`.

    Cela peut aider à améliorer la lisibilité et la maintenabilité des requêtes complexes.

    La syntaxe est :
    ```sql
    WITH cte_name AS (
      SELECT column_name
      FROM table_name
    )
    SELECT *
    FROM cte_name;
    ```
  </div>
  </slot>
</Challenge>
