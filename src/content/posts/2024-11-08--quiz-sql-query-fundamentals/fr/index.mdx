    {text: 'Utilisé uniquement pour définir des vues'},
    {text: 'Définir un ensemble de résultats temporaire à utiliser dans une instruction SQL plus grande', isAnswer: true},
  ]}
>
  <slot name="question">
  <div className="question">
    Quel est l'objectif de la clause `WITH` (Common Table Expression) en SQL ?
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    La clause `WITH`, ou Expression de Table Commun (CTE), est utilisée pour définir un ensemble de résultats temporaire pouvant être référencé dans la requête principale. Dans PostgreSQL, `WITH` peut être ajouté à des instructions comme `SELECT`, `INSERT`, `UPDATE`, `DELETE` ou `MERGE`.

    Cela peut améliorer la lisibilité et la maintenance des requêtes complexes.

    La syntaxe est :
    ```sql
        WITH cte_name AS (
        SELECT column_name
        FROM table_name
        )
        SELECT *
        FROM cte_name;
    ```
  </div>
  </slot>
