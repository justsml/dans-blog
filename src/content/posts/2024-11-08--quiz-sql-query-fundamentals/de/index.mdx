  ]}
>
  <slot name="question">
  <div className="question">
    Was ist der eigentliche Zweck der `WITH`-Klausel (Common Table Expression) in SQL?
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    Die `WITH`-Klausel, auch Common Table Expression (CTE) genannt, definiert ein temporäres Ergebnis-Set, das du innerhalb deiner Hauptabfrage direkt referenzieren kannst. In PostgreSQL lässt sich `WITH` an Anweisungen wie `SELECT`, `INSERT`, `UPDATE`, `DELETE` oder `MERGE` anhängen.

    Das bricht komplexe Logik in handliche Teile auf und verbessert die Lesbarkeit sowie Wartbarkeit deiner Queries erheblich.

    Die Syntax lautet:
    ```sql
    WITH cte_name AS (
    SELECT column_name
    FROM table_name
    )
    SELECT *
    FROM cte_name;
    ```
  </div>
  </slot>
</Challenge>
