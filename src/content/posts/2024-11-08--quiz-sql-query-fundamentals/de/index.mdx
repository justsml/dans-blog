  <div className="question">
    Welchem Zweck dient die `WITH`-Klausel (Common Table Expression) in SQL?
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    Die `WITH`-Klausel, auch als Common Table Expression (CTE) bezeichnet, dient dazu, einen temporären Ergebnisdatensatz zu definieren, der in der Hauptabfrage referenziert werden kann. In PostgreSQL kann `WITH` an Anweisungen wie `SELECT`, `INSERT`, `UPDATE`, `DELETE` oder `MERGE` angehängt werden.

    Dies kann die Lesbarkeit und Wartbarkeit komplexer Abfragen verbessern.

    Die Syntax ist:
    ```sql
        WITH cte_name AS (
        SELECT column_name
        FROM table_name
        )
        SELECT *
        FROM cte_name;
    ```
  </div>
  </slot>
