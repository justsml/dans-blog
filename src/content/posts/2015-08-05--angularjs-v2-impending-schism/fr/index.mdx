Je pense que nous assistons une fois de plus à la « conversion » Python 2→3.
AngularJS v2.0 introduit trop de changements. **Et TypeScript n'est pas le moindre**, ce qui est beaucoup demander alors que se finalise la dernière version de JS : ES6.

> Disons-le clairement : j'adore TypeScript. Je souhaitais secrètement que les réunions TC-39 l'aient produit... Ce n'est pas le cas.
> Cependant, ils ont proposé une autre spécification (totalement différente), tout aussi impressionnante...

Même si TypeScript se compile en JavaScript, cela ne signifie pas que vous copiez-collez aveuglément du TypeScript « compilé ». Il devient effectivement un apprentissage obligatoire, ne serait-ce que pour comprendre le TypeScript annoté d'AngularJS 2.0.

Désormais, les débutants doivent gravir le « Mont TypeScript » avant même de pouvoir commencer à assembler une application Angular (avec un minimum de compréhension).

J'ai une petite idée de la façon dont ça va se passer...
