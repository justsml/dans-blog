Promise の実行タイミングを可視化するために、`delay(millisecs)` という新しいメソッドを定義します。

```js
function delay(millisecs) {
  return new Promise(resolve => {
    setTimeout(() => resolve(millisecs), millisecs);
  });
}
```

このユーティリティは、タイムアウトが経過した時点で解決されます。

`delay` に渡したミリ秒数は、`.then` のコールバックへ引数として渡されます。

4 つの例（アニメーション付きタイムライン）を見てみましょう。

## Example #1/4

`console.log()` の実行が `delay(msec)` によって遅延される様子を示します。

```js
delay(1000).then(() => console.log("done"));
```

![Timeline showing delay 1000 then console log running after one second](../N_1000ms_log.webp)

{/* ```
delay(1000) --------|.then(fn)
                    | console.log('done')
|-------------------|--------------------|--------------------|-----------------
0msec             1sec                 2sec                 3sec
``` */}

## Example #2/4

_これは一般的な間違いの例です。_

`console.log` は `delay(1000)` が **開始** した瞬間に実行されます。遅延が **終了した後** に実行されるはずだったという期待とは逆です。

`console.log` の戻り値は `undefined` なので、`.then()` に渡された結果は無視され、何も起こりません。

`typeof console.log === 'function'` と `typeof console.log() === undefined` の違いに注意してください。

`console.log` の正しい使い方は例 #1 に示した通りです。`.then` や `.catch` には **関数** を渡すようにしましょう。

```js
delay(1000).then(console.log("done"));
```

![Timeline showing console log running immediately before the delay finishes](../N_1000ms_!log.webp)

{/* ```
delay(1000) --------|.then(null)
console.log('done')
|-------------------|--------------------|--------------------|-----------------
0msec             1sec                 2sec                 3sec
``` */}

## Example #3/4

3 つの Promise が同時に実行されます。

```js
delay(1000).then(console.log);
delay(2000).then(console.log);
delay(3000).then(console.log);
```

![Timeline showing three delay promises resolving after one two and three seconds](../N_3000ms.webp)

{/* ```
delay(1000) ------|.then(console.log)
delay(2000) ------|--------------------|.then(console.log)
delay(3000) ------|--------------------|--------------------|.then(console.log)
|-----------------|--------------------|--------------------|-------------------
|                 |                    |                    |
0msec           1sec                 2sec                 3sec
``` */}

## Example #4/4

`Promise.all` と 3 つの `delay` Promise。 同時に実行されます。

```js
Promise.all([delay(1000), delay(2000), delay(3000)]).then(console.log);
```
