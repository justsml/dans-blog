>
  <slot name="question">
  <div className="question">
    親/コンテナの幅が400pxの場合、次の要素の計算幅はいくつになりますか？
    ```css
        div {
          width: min(250px, 50%);
        }
    ```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    `min()` 関数は、250px と親の幅の50% のうち、最も小さい値を選択します。

    計算値を理解するには、相対単位をピクセルに変換する必要があります:

    - `400px` の `50%` は `200px`
    - `250px` はすでにピクセル単位です
    ```css
    /* This gets computed to */
    width: min(250px, 200px);
    /* -> 200px wins */
    ```
    `min()` 関数はレスポンシブデザインに特に便利で、コンポーネント（またはフォントサイズ）が特定の制限を超えないようにすることができます。
  </div>
  </slot>
</Challenge>

    - `50px`はすでにピクセルです

    - `6rem`は`6 * 16px`（デフォルトフォントサイズ）で`96px`です
    ```css
    /* This gets computed to */
    width: max(50px, 20px, 96px);
    /* -> 96px wins */
    ```

  <slot name="question">

    ```css
        .card {

        }
    ```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    `clamp()`関数は、`50vw`に基づいて幅をスケーリングできますが、200pxから500pxの範囲内に収めます。

    つまり、50vwが200px未満の場合は幅が200px、50vwが500pxを超える場合は500pxとなり、その間は線形に変化します。

    これにより、自動的にレスポンシブになります！`clamp`について知っておくべきことは、**固定単位**と**レスポンシブまたは計算単位**を組み合わせている点です。

    通常、フォントサイズにビューポート単位を使いたくはありませんが、`clamp()`を使えばフォントサイズが小さくなりすぎたり大きくなりすぎたりするのを防げます。
  </div>
  </slot>
