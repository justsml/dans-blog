  options={[

    {text: 'width: 500px;'},

>
  <slot name="question">
  <div className="question">
Wenn die Breite des übergeordneten Containers 400px beträgt, wie groß ist die berechnete Breite des folgenden Elements?
```css
    div {
      width: min(250px, 50%);
    }
```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
Die `min()`-Funktion wählt strikt den kleinsten Wert zwischen `250px` und `50%` der Breite des Elternelements.

Damit du den berechneten Wert nachvollziehen kannst, rechnen wir die relativen Einheiten fix in Pixel um:

- `50%` von `400px` ergibt `200px`
- `250px` steht bereits in Pixel
```css
/* This gets computed to */
width: min(250px, 200px);
/* -> 200px wins */
```
Im responsiven Design ist `min()` ein echtes Arbeitstier: Damit garantierst du, dass eine Komponente (oder Schriftgröße) niemals über ein definiertes Limit hinauswächst.
  </div>
  </slot>
</Challenge>

Ein Container ist `200px` breit. Wie groß ist die berechnete Breite des `<div>`?
```css
    div {
      width: max(50px, 10%, 6rem);
    }
```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
Die `max()`-Funktion akzeptiert zwei oder mehr Werte und wählt automatisch den größten aus. Geht man von der browserüblichen Standard-Schriftgröße von `16px` aus, ergibt sich eine Breite von `96px`.

Um den berechneten Wert nachzuvollziehen, müssen wir die relativen Einheiten in Pixel umrechnen:

- `50px` ist bereits in Pixeln angegeben
- `10%` von `200px` entspricht `20px`
- `6rem` ist `6 * 16px` (die Standard-Schriftgröße), also `96px`
```css
/* This gets computed to */
width: max(50px, 20px, 96px);
/* -> 96px wins */
```

  </slot>

```css

    }
```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
Die `clamp()`-Funktion lässt die Breite basierend auf `50vw` skalieren, hält sie aber strikt im Bereich von 200px bis 500px.

Konkret heißt das: Die Breite bleibt bei 200px, wenn 50vw darunter fallen würde, springt auf 500px, wenn 50vw darüber liegt, und skaliert dazwischen linear.

Damit wird dein Layout quasi automatisch responsiv! Der Clou bei `clamp` ist, dass es **feste Einheiten** mit **responsiven oder berechneten Einheiten** kombiniert.

Normalerweise würdest du Viewport-Einheiten nicht für Schriftgrößen verwenden wollen, aber mit `clamp()` garantierst du, dass die Schrift nie zu winzig oder zu riesig wird.

CSS-Variablen sind veränderbar und lassen sich zur Laufzeit direkt per JavaScript anpassen.
