    La clave aquí es <b>actualizar</b>, no insertar ni hacer PUT. Si estás realizando inserciones, puedes usar `BatchWriteItem` o `TransactWriteItems`.

    Aunque `BatchWriteItem` puede manejar múltiples operaciones, está limitado a PUTs y DELETES. `TransactWriteItems` es más potente, pero es un poco un martillo de demolición para actualizaciones simples.

    La operación `UpdateItem`:
    - Actualiza atributos de un elemento existente.
    - Añade nuevos atributos a un elemento existente.
    - Elimina atributos de un elemento existente.
    - Realiza la actualización condicional si el elemento existe o cumple ciertas condiciones.
