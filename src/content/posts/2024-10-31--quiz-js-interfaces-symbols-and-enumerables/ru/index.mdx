  <div className="question">
    Будет ли свойство `age` перечислено при итерации `for...in`?
    ```js
        const person = {};
        Object.defineProperty(person, 'age', {
          value: 25,
          enumerable: true
        });
        for (let key in person) {
          console.log(key);
        }
    ```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    Свойство `enumerable` в `Object.defineProperty()` определяет, будет ли свойство отображаться в методах перечисления, таких как `for...in`. В данном примере, так как `enumerable: true`, свойство `age` будет перечислено при итерации.
  </div>
  </slot>
</Challenge>

    ```js
        const car = {};
        Object.defineProperty(car, 'make', {
          value: 'Toyota'
        });
        console.log(Object.keys(car));
    ```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    При использовании `Object.defineProperty()` без указания `enumerable` его значение по умолчанию — `false`. Это означает, что свойство `make` не будет отображаться в `Object.keys()` или других методах перечисления.
  </div>
  </slot>
</Challenge>
