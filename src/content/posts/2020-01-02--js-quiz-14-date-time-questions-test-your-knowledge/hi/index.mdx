## `Date` क्लास को आप कितना अच्छे से जानते हैं?

> * **अपनी JavaScript स्किल्स साबित करें!** 🚀
> * लॉगिन या साइनअप की कोई ज़रूरत नहीं। ✨
> * मल्टीपल चॉइस। 🤖 ... _कितना मुश्किल हो सकता है, है ना?_

### रूपरेखा

JavaScript की `Date` क्लास का API कुख्यात रूप से जटिल है। यह Java से विरासत में मिला है, और मेरा अंदाज़ा है कि शायद यह प्राचीन नवपाषाण काल के समय-मापन तरीकों से प्रेरित था।

`Date` को सही से हैंडल करने का संघर्ष अक्सर डेवलपर्स को बिना सवाल किए थर्ड-पार्टी लाइब्रेरीज़ अपनाने पर मजबूर कर देता है। हालाँकि ये अक्सर सुरक्षित और भरोसेमंद विकल्प होते हैं, लेकिन डेट फॉर्मेटिंग या लोकलाइज़ेशन के लिए इनकी शायद ही कभी ज़रूरत पड़ती है!

यह क्विज़ नेटिव `Date` API के बारे में आपके ज्ञान को परखने (और गहरा करने) के लिए डिज़ाइन किया गया है। हिंट और एक्सप्लेनेशन के लिए हरे बटनों का उपयोग करें! उम्मीद है कि इस चैलेंज के अंत तक JavaScript में `Date` को लेकर आपकी समझ मज़बूत हो जाएगी।
