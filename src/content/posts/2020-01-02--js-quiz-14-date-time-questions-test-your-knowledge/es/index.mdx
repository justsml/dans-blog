## ¿Qué tan bien conoces la clase `Date`?

### Estructura

La clase `Date` de JavaScript tiene una API notoriamente difícil. Se heredó de Java y solo puedo asumir que se inspiró en métodos neolíticos de medición del tiempo.

La lucha por hacer que `Date` funcione lleva a muchos desarrolladores a usar bibliotecas de terceros sin cuestionarlo. Aunque suele ser una opción segura y fiable, ¡estas bibliotecas rara vez son necesarias para formatear fechas o localizarlas!

Este cuestionario está diseñado para poner a prueba (y profundizar) tu conocimiento de la API nativa `Date`. ¡Usa los botones verdes para obtener pistas y explicaciones! Con suerte, al final del desafío habrás consolidado tu comprensión de `Date` en JavaScript.

      El formato `T00:00` hace que el segundo valor se interprete como la medianoche local.

      La primera fecha se interpreta como `Tue Dec 31 2019 17:00:00 GMT-0700 (Mountain Standard Time)`.
