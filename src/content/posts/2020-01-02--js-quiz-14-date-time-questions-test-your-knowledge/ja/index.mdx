## `Date` クラスをどのくらい理解しているか？

> * **JavaScript の腕前を証明しろ！** 🚀
> * ログインやアカウント登録は不要。✨
> * 択一問題。🤖 ... _どれほど難しいというんだ？_

### 概要

JavaScript の `Date` クラスは、悪名高いほど扱いにくい API を持っている。Java から継承されたもので、おそらく古代新石器時代の時刻計測方法にインスパイアされたのだろうと推測するしかない。

`Date` の仕様と格闘する苦労から、多くの開発者は疑問を持たずにサードパーティ製ライブラリに頼りがちだ。確かに安全で信頼性の高い選択肢ではあるが、日付のフォーマットやローカライズだけであれば、それらのライブラリはほとんど必要ない！

このクイズは、ネイティブの `Date` API に関する知識を試す（そして深める）ために設計されている。ヒントや解説には緑色のボタンを使ってくれ。チャレンジを終える頃には、JavaScript の `Date` に関する理解が確実に固まっているはずだ。
