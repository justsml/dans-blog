    背压是一种机制，当写入端跟不上时暂停读取，从而防止内存溢出。

    手动背压示例：
    ```js
    readable.on('data', (chunk) => {
      const canContinue = writable.write(chunk);
      if (!canContinue) {
        readable.pause();
        writable.once('drain', () => readable.resume());
      }
    });
    ```
    `pipe()` 会自动处理！

    [了解更多关于背压的信息](https://nodejs.org/en/learn/modules/backpressuring-in-streams#backpressuring-in-streams)
