### Исправление disable-transparent-hugepages для Debian/Ubuntu

> Появляется предупреждение "WARNING: /sys/kernel/mm/transparent_hugepage/defrag is 'always'." ?

Выполните следующие команды, чтобы быстро сделать то, что [MongoDB описывает подробнее](https://docs.mongodb.org/v3.0/tutorial/transparent-huge-pages/).

```sh
## Currently just debian
sudo curl -sSL -o /etc/init.d/disable-transparent-hugepages https://gist.githubusercontent.com/justsml/5e8f10892070072c4ffb/raw/disable-transparent-hugepages
sudo chmod 755 /etc/init.d/disable-transparent-hugepages
sudo update-rc.d disable-transparent-hugepages defaults
```
