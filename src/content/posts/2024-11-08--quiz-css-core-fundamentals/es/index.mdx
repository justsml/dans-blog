>
  <slot name="question">
  <div className="question">
¿Qué selector coincide mejor con el siguiente HTML?
```html
      ```html
      <a id="home" name="home" href="/home">Home</a>
      ```
```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
La respuesta correcta es `a#home[name='home']`, ya que apunta específicamente a los atributos `id` y `name`. Recuerda que los selectores CSS distinguen mayúsculas de minúsculas, así que `#Home` fallaría. Por otro lado, un espacio entre selectores indica una relación de padre a hijo, lo cual no aplica aquí.

El pseudo-selector `:contains()` no es parte del estándar oficial de CSS, aunque algunas librerías de JavaScript sí lo soportan.
  </div>
  </slot>
