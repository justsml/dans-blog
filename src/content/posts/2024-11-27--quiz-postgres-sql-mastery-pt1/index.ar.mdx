  <div className="explanation">
    تقبل PostgreSQL ثلاث أنماط للتحويل:

    1. SQL القياسي: `CAST(expression AS type)`.
    2. PostgreSQL: `expression::type`.
    3. دالة التحويل: `type 'literal'`.

    جميعها متطابقة في الوظيفة، لكن:
    - `CAST()` هي أكثر قابلية للتنقل.
    - `::` هو خاص بPostgreSQL ولكنه شائع الاستخدام.
    - التعبير المترادف `type 'literal'` أقل شيوعًا ولكن لا يزال صحيحًا.

    - تسمح بعدة قيم NULL.

    للحظر القيم `NULL`، أضف `NOT NULL`:
    ```sql
    CREATE TABLE student_emails (
      student_id INTEGER,
      email VARCHAR(255) NOT NULL,
      UNIQUE(email)
    );
    ```
