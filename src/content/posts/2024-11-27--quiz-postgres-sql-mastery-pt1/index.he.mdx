    1. ANSI SQL: `CAST(expression AS type)`.
    2. PostgreSQL: `expression::type`.
    3. מנגנון בינארי: `type 'literal'`.

    - `CAST()` הוא המקומי ביותר
    - `::` הוא מ-PostgreSQL אבל נפוץ
    - `type 'literal'` פחות נפוץ אבל עדיין חוקי

    ```sql
    );
    ```
  </div>

  <div className="explanation">
    Constraints אל-UNIQUE ב-PostgreSQL:
    - מורשעים מספרי NULL רבים
    - `NULL` ≠ `NULL` בבדיקות האוניברסיטה

    למניעת ערכים של `NULL`, הוסף `NOT NULL`:
    ```sql
    CREATE TABLE student_emails (
      student_id INTEGER,
      email VARCHAR(255) NOT NULL,
      UNIQUE(email)
    );
    ```
