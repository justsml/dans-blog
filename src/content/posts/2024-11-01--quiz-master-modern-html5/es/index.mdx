El elemento [`<meter>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/meter) se utiliza para representar una medición escalar (única) dentro de un rango establecido, como la temperatura, el uso de disco o un recuento de votos.

Puede parecer similar a una barra de [`<progress>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/progress), sin embargo, las barras de progreso **SIEMPRE** comienzan en cero. Por lo tanto, los elementos `<progress>` muestran un `porcentaje de finalización`, mientras que un `<meter>` muestra cualquier valor dentro de un rango definible.
```html
```html
<meter min="-60" max="130" value="75" /> 75°F
<meter min="0" max="100" value="75" /> 75%
```
