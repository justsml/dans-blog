## Componenti UI, blocchi e template

Questa è la sezione che la maggior parte delle persone vuole davvero: template di siti web, hero section, app shell, blocchi Tailwind e componenti UI da adattare invece di progettare da una tela bianca.

### UI AI e chat

- [Assistant UI](https://www.assistant-ui.com/) — Componenti UI open-source per chat AI in app React.
- [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought) — Un pattern pulito di chat UI per prodotti AI.

### Blocchi, sezioni e componenti

- [Aceternity UI](https://ui.aceternity.com/components) — Componenti React e Tailwind ad alta precisione con forte personalità visiva.
- [reui Collapsible](https://reui.io/components/collapsible) — Pattern UI collapsible e accordion puliti per interfacce React.
- [Shadcnblocks](https://www.shadcnblocks.com/blocks) — Sezioni e blocchi pronti basati sull'ecosistema shadcn.
- [Tailark](https://tailark.com/blocks?licences=free) — Blocchi Tailwind gratuiti per siti marketing e pagine prodotto.
- [Tool UI](https://www.tool-ui.com/) — Blocchi UI e componenti moderni per app e landing page.
- [CSS Layout](https://csslayout.io/patterns) — Pattern di layout copy-paste per strutture comuni di siti web.

### Template completi e ispirazione

- [HTML5 UP](https://html5up.net) — Template HTML belli e responsive.
- [Creative Tim Free React Templates](https://www.creative-tim.com/templates/react-free) — Dashboard React gratuite, kit e template admin.
- [Awesome Landing Page](https://github.com/nordicgiant2/awesome-landing-page) — Template e esempi curati di landing page.

- [The Stocks](http://thestocks.im/) — Aggrega font, icone, foto e altre risorse creative.

### Colore e contrasto

- [Adobe Color](https://color.adobe.com/create/color-wheel/) — Ruota dei colori classica e strumenti di armonia.
- [Coolors](https://coolors.co/) — Genera, blocca e regola palette velocemente.
- [Color Hunt](https://colorhunt.co/) — Ispirazione per palette curate.
- [Paletton](https://paletton.com/) — Generatore di schemi colore con anteprime.

- [Contrast Ratio](https://contrast-ratio.com/) — Test rapido del contrasto con un'interfaccia più pulita.

## Animazione e interazione

Usali quando i layout statici sembrano piatti e vuoi motion migliori, stati hover o navigazione più viva.

- [GSAP](https://greensock.com/gsap) — Il cavallo di battaglia del settore per animazioni UI complesse.
- [Anime.js](https://animejs.com/) — Motore di animazione JavaScript potente con una curva di apprendimento più amichevole.
- [Animate.css](https://animate.style/) — Animazioni preset veloci per ingressi semplici e feedback.
- [Hover.css](https://ianlunn.github.io/Hover/) — Interazioni hover e micro-effetti pronti.
- [Magic Animations](https://www.minimamente.com/project/magic/) — Animazioni CSS preset.
- [Bounce.js](https://bouncejs.com/) — Generazione di keyframe con effetto spring. Archiviato, ma ancora utile per ispirazione.
- [Hamburgers](https://github.com/jonsuh/hamburgers) — Animazioni accessibili per menu hamburger.
- [Best Snip SVG Animator](https://bestsnip.com/svg_animation/) — Semplice strumento web per animare SVG.
- [KokonutUI: Flow Field](https://kokonutui.com/docs/backgrounds/flow-field) — Ricetta per sfondi animati con un forte impatto visivo.
- [KokonutUI: Smooth Tab](https://kokonutui.com/docs/navigation/smooth-tab) — Pattern di navigazione tab smooth da adattare per UI di app.

- [Iconoir](https://iconoir.com) — Grande set di icone SVG gratuite con un look leggermente più morbido.
- [Simple Icons](https://simpleicons.org/) — Icone di brand per prodotti e link social.

## Suggerimenti su licenze e utilizzo

- Non dare per scontato che "gratuito" significhi "nessuna attribuzione." Controlla ogni licenza prima di spedire lavoro per clienti.
- Fai attenzione alle restrizioni di trademark quando usi marchi, loghi o icone di app in template.
- Esporta in SVG quando possibile così icone e illustrazioni restano nitide e più facili da tematizzare.
- Mantieni le famiglie di icone coerenti. Un'icona Lucide pulita accanto a un SVG cartoonish da marketplace di solito sembra accidentale.
- Testa sempre il contrasto del testo. Molte palette "bellissime" crollano sul body copy reale.
- Tratta le illustrazioni stock come materiale grezzo. Un po' di editing fa molta strada per evitare quella sensazione di template copiato.
