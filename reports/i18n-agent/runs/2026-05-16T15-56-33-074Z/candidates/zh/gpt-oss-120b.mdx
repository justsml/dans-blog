## 问题：人不是边界，人是流量
人不是边界。人是流量。在 AI 驱动的攻击世界中，你的笔记本电脑就是最高奖赏。

### 本地制品对比
| 特性 | 标准笔记本电脑 | Dev Container |
| :--- | :--- | :--- |
| 凭据隔离 | 低 | 高 |
| 响应速度 | 慢 | 即时 |

## 解决方案：Dev Containers 与 Canarytokens
通过使用 Dev Containers，我们隔离了开发工具。通过使用 Canarytokens，我们在几秒钟内检测到入侵。

```json
// .devcontainer/devcontainer.json
{
  "name": "Secure Environment",
  "features": {
    "ghcr.io/devcontainers/features/sshd:1": {}
  }
}
```

## 结论
安全不是一种状态，而是一个快速响应的过程。
