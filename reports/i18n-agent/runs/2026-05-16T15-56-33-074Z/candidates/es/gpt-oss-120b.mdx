# En la brecha

Las laptops de los desarrolladores son almacenes de credenciales. Si una se ve comprometida, la cadena de suministro completa está en riesgo.

## El Problema: Humanos como Tráfico
Los humanos no son límites. Los humanos son tráfico. En un mundo de ataques impulsados por IA, su laptop es el premio mayor.

### Comparación de Artefactos Locales
| Característica | Laptop Estándar | Dev Container |
| :--- | :--- | :--- |
| Aislamiento de Credenciales | Bajo | Alto |
| Velocidad de Respuesta | Lenta | Instantánea |

## Solución: Dev Containers y Canarytokens
Al usar Dev Containers, aislamos las herramientas de desarrollo. Al usar Canarytokens, detectamos la brecha en segundos.

```json
// .devcontainer/devcontainer.json
{
  "name": "Secure Environment",
  "features": {
    "ghcr.io/devcontainers/features/sshd:1": {}
  }
}
```

## Conclusión
La seguridad no es un estado, es un proceso de respuesta rápida.
