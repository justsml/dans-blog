## Docker es genial

> Actualizaciones septiembre 2016, 2018
> Boot2Docker ha sido reemplazado por Docker para Mac

> Nota histórica: esta publicación se conserva intencionalmente como una instantánea de Docker para Mac de la era 2015. Los consejos de rendimiento y los nombres de herramientas reflejan ese momento; los flujos de trabajo actuales de Docker Desktop y Compose han evolucionado.

1. Docker es increíble, rápido y flexible.
1. Las herramientas anteriores, especialmente boot2docker, eran lentas y propensas a fallar.

Docker puede ejecutarse actualmente de forma nativa en un kernel Linux v3.4+ - y la máquina virtual boot2docker actual ejecuta v4.

Mejor uso de tu hardware: instala la última versión de Debian o Ubuntu en tu Mac/Windows,

... vamos, esos juegos no están ayudando a tu código...

### Revisa tu configuración

Revisa la salida del comando `docker info`.

1. Seguridad: Verifica que el servidor no tenga puertos abiertos inesperados (con `nmap` desde una red remota)
1. DNS: Usa una caché local o un servidor DNS de baja latencia.
1. Almacenamiento: Usa el controlador de almacenamiento correcto (`overlay2` es probable)

Actualizado 2024:

- Docker Desktop es propietario, pero gratuito para uso personal. Es una excelente manera de empezar con Docker en OSX o Windows.
- Si buscas una solución más abierta, echa un vistazo a [Rancher Desktop](https://rancherdesktop.io/).
