## प्रोग्रामिंग भाषाओं के नोट्स

#### मैं बिल्कुल निश्चित नहीं हूँ कि मेरे यादृच्छिक अवलोकन पहले से किए गए हैं, लेकिन यह मेरी सबसे दिलचस्प भाषाओं की सूची है:

### जावास्क्रिप्ट

मेरी एक सच्ची प्रेमिका, अत्यंत बहुमुखी और व्यापक - सर्वांगीण, शानदार शक्तिशाली चैंपियन!
यह GitHub.com पर लगातार वर्षों से सबसे सक्रिय/लोकप्रिय भाषा है।

मैंने इसे मानना पड़ा कि लंबे समय तक मैंने बेवजह उपेक्षा और अपमान के साथ केवल यह भाषा देखा, जो अब **मेरी पसंदीदा भाषा** है।

**ES6** ने मेरी ~~~आदिक्षा~~~ प्रेम को और बढ़ा दिया। जबकि शुद्ध ES5 हमेशा मेरे दिल में विशेष स्थान रखेगा, हर बार जब मैंने कुछ **ES6** का उपयोग किया, तो मुझे वह विकिरणित मकड़ी के काटने की भावना महसूस हुई...

मुझे **ES6 के तरफ धकेलने वाले 4 कारण थे**:

1.  यह मज़ेदार है। गंभीरता से। सौंदर्य, स्पष्टता और उत्पादकता में ठोस लाभ हैं।

- आप कहेंगे कि यह विषयगत दावे हैं? मैं आपको कुछ ES6 के उदाहरण दिखाता हूँ:
- `let expired = users.filter(u => Date.now() > u.trialDate)`
- अब आपको यह झूठ बोलने की आवश्यकता नहीं है कि आपको `Object.create` या `Object.defineProperty` का उपयोग कैसे करना है
- नीचे दिए गए उदाहरण देखें

1. जुलाई 2015 तक, ES6 आधिकारिक रूप से अब अंतिम रूप ले चुका है!  
1. समर्थन प्रभावी रूप से 100% है! ... ठीक है, BabelJS की आवश्यकता होती है ताकि आपकोड को ES5 संगत बनाए रखें। ऐतिहासिक रूप से JS ट्रांसपाइलर्स का अवरोध होता रहा है। हालांकि, अब (2014-15) से बदलाव हो गया है क्योंकि BabelJS भाषा विकास के महत्वपूर्ण सक्रमणकारी/चालक बन गया है। माइक्रोसॉफ्ट और फेसबुक सहित बड़ी संख्या में कंपनियां इसका उपयोग सबसे बड़ी वेबसाइटों पर करती हैं।  
1. [नोड के नवीनतम संस्करण](https://nodejs.org/en/blog/release/v4.0.0/) चल रहे हैं, जिनमें चार्म v45 के समान V8 JS इंजन है, यह v4.5 है  

#### उदाहरण  

> मैं आपको दिखाऊंगा कि अंततः मुझे _शुरू_ करने के लिए कैसे ES6-स्वाद का KoolAid पीना पड़ा।  

मेरे हालिया अनुभव में, ES6 आपको तेजी से कोड लिखने में मदद करता है। इतना कि कोड अधिक संक्षिप्त होने के कारण, आपको अपने पुराने कोड (या अपने टीम के साथी के कोड) को समझने के लिए कम मस्तिष्क शक्ति की आवश्यकता होती है।  

मैंने नियमित रूप से KLOC बचत देखी है जो लगभग 20-50% है। यह केट मॉस के तरह पतला है!  

**_अपलोड नहीं हुई छवि:_ EcmaScript 5 vs ES 2016 - डेमो: क्लास, डिस्ट्रक्चरिंग, स्लिक**  
{/* ](/images/screenshots/Diff-ES6-vs-ES5-Head-to-Head.png) */}  

```js  
// /services/users.js  
class Users {  
  constructor(data) {  
    this.users = data || [];  
  }  
  expired() {  
    return this.users.filter(u => Date.now() > u.trialDate);  
  }  
}  
```  

- अब कोई भी जरूरत नहीं है कि किसी फ़ंक्शन में पारित किए गए मैदानों को 'निकालें' और 'जांचें' करने के लिए। उदाहरण `add()` देखें:  

```js  
// /services/users.js  
class Users {  
  constructor(data) {  
    this.users = data || [];  
  }  
  add({ name, email, password }) {  
    // पासवर्ड हैश संग्रहीत करें, हमें केवल 1 स्पष्ट `var/let` की आवश्यकता है - अन्य वैरिएबल्स को `{fields}` जादू के ऊपर ^^^ के साथ 'परिभाषित' किया गया है  
    let hash = getSha256(password);  
    return http  
      .post("/users", {  
        name: name,  
        email: email,  
        passwordHash: hash  
      })  
      .then(usr => this.users.push(usr)); // सेवा प्रतिक्रिया पर उपयोगकर्ता को जोड़ें  
  }  
}  
```

  - लिखने में बहुत मजेदार! समझौते और अपघटन अच्छी विशेषताएँ हैं और अन्य भाषाओं को बस बोझिल बना देते हैं!
  - सरणियाँ, 'अनुक्रम', ट्यूपल आदि अपेक्षाकृत सरल हैं

```python
## dummy code: defines a color + pixel-coord -
def pixel(x, y, r, g, b): return dict(x=x, y=y, r=r, g=g, b=b)
## Create a new pixel object and apply to set of vars
x, y, r, g, b = pixel(10, 20, 255, 255, 255)
## Now we can call pixel
```
