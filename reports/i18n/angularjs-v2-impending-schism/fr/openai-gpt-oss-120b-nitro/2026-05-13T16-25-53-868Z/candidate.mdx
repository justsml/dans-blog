Je penseque nous revivons la « Conversion » Python 2→3.  
AngularJS v2.0 introduit trop de changements. **Pas le moindre étant TypeScript**, ce qui représente une grosse demande alors que la dernière version de JS, ES6, est en cours de finalisation.

> Dites-le clairement : j’aime TypeScript. J’aimerais secrètement que les réunions TC‑39 l’aient produit… Ce n’est pas le cas.  
> Cependant, ils ont proposé une autre spécification (tout à fait différente), tout aussi géniale…

Bien que TypeScript se compile en JavaScript, cela ne signifie pas que vous pouvez copier‑coller aveuglément le TypeScript « compilé ». Cela devient effectivement une compétence obligatoire pour comprendre le TypeScript annoté d’AngularJS 2.0.

Ainsi, les novices doivent gravir le « Mont TypeScript » avant même de pouvoir commencer à assembler une application Angular (avec un minimum de compréhension).

J’ai le pressentiment que cela va se passer ainsi…
