Je pense que nous assistons à la conversion Python 2 vers 3 une fois de plus.  
AngularJS v2.0 introduit trop de changements. **Pas le moindre desquels est TypeScript**, ce qui représente une demande ambitieuse alors que la version la plus récente de JS, ES6, est en cours de finalisation.

> Laissez-moi être clair : j'adore TypeScript. Je souhaitais secrètement que les réunions du TC-39 aient produit TypeScript... Elles ne l'ont pas fait.  
> Cependant, elles ont élaboré une autre spécification (totalement différente), tout aussi formidable...

Bien que TypeScript se compile en JavaScript, cela ne signifie pas que vous pouvez copier-coller aveuglément le TypeScript « compilé ». Il devient effectivement une compétence requise pour comprendre le TypeScript annoté d'AngularJS 2.0.

Ainsi, les débutants doivent gravir « Mount TypeScript » avant même de pouvoir commencer à assembler une application Angular (avec un certain niveau de compréhension).
