    ```js
        const car = {};
        Object.defineProperty(car, 'make', {

        });
        console.log(Object.keys(car));
    ```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    `Object.defineProperty()`で`enumerable`を明示的に指定しない場合、デフォルトは`false`です。このため`make`プロパティは`Object.keys()`や他の列挙メソッドで表示されません。
  </div>
  </slot>

