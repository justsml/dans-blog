## 从这里开始

如果你只需要几个可靠的选择，从这些开始：

- 精致的 React 或 Tailwind UI 区块：[Aceternity UI](https://ui.aceternity.com/components)、[Shadcnblocks](https://www.shadcnblocks.com/blocks) 和 [Tailark](https://tailark.com/blocks?licences=free)。
- AI 产品界面与聊天模式：[Assistant UI](https://www.assistant-ui.com/) 和 [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought)。
- 图标：[Lucide](https://lucide.dev/)、[Iconoir](https://iconoir.com) 和 [Simple Icons](https://simpleicons.org/)。
- 插画：[ManyPixels](https://www.manypixels.co/gallery/)、[DrawKit Free](https://www.drawkit.io/free) 和 [Humaaans](https://www.humaaans.com/)。
- 字体：[Fontsource](https://fontsource.org) 和 [Google Fonts](https://fonts.google.com/)。
- 色彩与无障碍：[Coolors](https://coolors.co/)、[Adobe Color](https://color.adobe.com/create/color-wheel/) 和 [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)。

## UI 组件、区块和模板

这是大多数人真正想要的部分：网站模板、英雄区域、应用外壳、Tailwind 区块，以及你可以直接改造而非从零设计的 UI 组件。

### AI 与聊天 UI

- [Assistant UI](https://www.assistant-ui.com/) — 用于 React 应用的开源 AI 聊天 UI 组件。
- [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought) — 适用于 AI 产品的简洁推理式聊天 UI 模式。

### 区块、片段与组件

- [Aceternity UI](https://ui.aceternity.com/components) — 高度精致的 React 和 Tailwind 组件，视觉风格鲜明。
- [reui Collapsible](https://reui.io/components/collapsible) — 用于 React 界面的简洁折叠与手风琴式 UI 模式。
- [Shadcnblocks](https://www.shadcnblocks.com/blocks) — 围绕 shadcn 生态构建的现成区块和片段。
- [Tailark](https://tailark.com/blocks?licences=free) — 适用于营销网站和产品页面的免费 Tailwind 区块。
- [Tool UI](https://www.tool-ui.com/) — 适用于应用和落地页的现代 UI 区块与组件。
- [CSS Layout](https://csslayout.io/patterns) — 常见网站结构的可复制粘贴布局模式。

### 完整模板与灵感

- [HTML5 UP](https://html5up.net) — 美观、响应式的 HTML 模板。
- [Creative Tim Free React Templates](https://www.creative-tim.com/templates/react-free) — 免费的 React 仪表盘、工具包和管理类模板。
- [Awesome Landing Page](https://github.com/nordicgiant2/awesome-landing-page) — 精选的落地页模板与示例。

- [Penpot](https://penpot.app/) — 开源设计与原型工具。

## 字体与色彩工具

- [Google Fonts](https://fonts.google.com/) — 经典的字体目录。
- [Fontpair](https://fontpair.co/fonts/) — 当你需要快速决策时的字体搭配灵感。
- [The Stocks](http://thestocks.im/) — 聚合字体、图标、照片及其他创意资源。

### 色彩与对比度

- [Adobe Color](https://color.adobe.com/create/color-wheel/) — 经典色轮与和谐工具。
- [Coolors](https://coolors.co/) — 快速生成、锁定和调整调色板。
- [Color Hunt](https://colorhunt.co/) — 精选调色板灵感。
- [Paletton](https://paletton.com/) — 配色方案生成器，带预览。

- [Contrast Ratio](https://contrast-ratio.com/) — 快速对比度测试，界面更简洁。

## 动画与交互

当静态布局显得平淡，你需要更好的动效、悬停状态或更生动的导航时，使用这些工具。

- [GSAP](https://greensock.com/gsap) — 复杂 UI 动画的行业主力工具。
- [Anime.js](https://animejs.com/) — 功能强大的 JavaScript 动画引擎，学习曲线更友好。
- [Animate.css](https://animate.style/) — 快速预设动画，用于简单的入场和反馈。
- [Hover.css](https://ianlunn.github.io/Hover/) — 现成的悬停交互和微效果。
- [Magic Animations](https://www.minimamente.com/project/magic/) — 预设 CSS 动画。
- [Bounce.js](https://bouncejs.com/) — 类似弹簧的关键帧生成。已归档，但仍可用于灵感参考。
- [Hamburgers](https://github.com/jonsuh/hamburgers) — 无障碍的汉堡菜单动画。
- [Best Snip SVG Animator](https://bestsnip.com/svg_animation/) — 简单的 SVG 动画网页工具。
- [KokonutUI: Flow Field](https://kokonutui.com/docs/backgrounds/flow-field) — 具有强烈视觉效果的动画背景方案。
- [KokonutUI: Smooth Tab](https://kokonutui.com/docs/navigation/smooth-tab) — 平滑标签导航模式，可适配应用 UI。

- [Iconoir](https://iconoir.com) — 大型免费 SVG 图标集，风格稍显柔和。

## 许可与使用提示

- 不要假设“免费”意味着“无需署名”。在交付客户项目前，务必检查每项许可。
- 在模板中使用品牌标识、Logo 或应用图标时，注意商标限制。
- 尽可能导出 SVG，这样图标和插画保持清晰，且更易于主题化。
- 保持图标系列的一致性。一个干净的 Lucide 图标旁边放一个卡通风格的市场 SVG 图标，通常看起来像是拼凑的。
- 始终测试文本对比度。许多“漂亮”的调色板在真实正文中会失效。
- 将图库插画视为原材料。稍加编辑就能大大避免那种复制模板的感觉。
