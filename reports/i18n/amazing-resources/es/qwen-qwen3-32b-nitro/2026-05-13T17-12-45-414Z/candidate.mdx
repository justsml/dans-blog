## Comience aquí

Si solo necesita unas pocas opciones confiables, comience con estas:

- Para bloques de UI de React o Tailwind pulidos: [Aceternity UI](https://ui.aceternity.com/components), [Shadcnblocks](https://www.shadcnblocks.com/blocks) y [Tailark](https://tailark.com/blocks?licences=free).
- Para interfaces de productos de IA y patrones de chat: [Assistant UI](https://www.assistant-ui.com/) y [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought).
- Para iconos: [Lucide](https://lucide.dev/), [Iconoir](https://iconoir.com) y [Simple Icons](https://simpleicons.org/).
- Para ilustraciones: [ManyPixels](https://www.manypixels.co/gallery/), [DrawKit Free](https://www.drawkit.io/free) y [Humaaans](https://www.humaaans.com/).
- Para fuentes: [Fontsource](https://fontsource.org) y [Google Fonts](https://fonts.google.com/).
- Para color y accesibilidad: [Coolors](https://coolors.co/), [Adobe Color](https://color.adobe.com/create/color-wheel/) y [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/).

## Componentes, bloques y plantillas de UI

Esta es la sección que más gente busca realmente: plantillas de sitio web, secciones hero, caparazones de aplicación, bloques de Tailwind y componentes de UI que puede adaptar en lugar de diseñar desde una hoja en blanco.

### AI y UI de chat

- [Assistant UI](https://www.assistant-ui.com/) — Componentes de interfaz de chat de IA de código abierto para aplicaciones de React.  
- [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought) — Un patrón de interfaz de chat estilo razonamiento para productos de IA.  

### Bloques, secciones y componentes  

- [Aceternity UI](https://ui.aceternity.com/components) — Componentes de React y Tailwind de alta calidad con fuerte personalidad visual.  
- [reui Collapsible](https://reui.io/components/collapsible) — Patrones de interfaz plegables y estilo acordeón para interfaces de React.  
- [Shadcnblocks](https://www.shadcnblocks.com/blocks) — Secciones y bloques listos para usar construidos en torno al ecosistema shadcn.  
- [Tailark](https://tailark.com/blocks?licences=free) — Bloques de Tailwind gratuitos para sitios de marketing y páginas de productos.  
- [Tool UI](https://www.tool-ui.com/) — Bloques y componentes modernos para aplicaciones y páginas de aterrizaje.  
- [CSS Layout](https://csslayout.io/patterns) — Patrones de diseño para estructuras comunes de sitios web que se pueden copiar y pegar.  

### Plantillas completas e inspiración  

- [HTML5 UP](https://html5up.net) — Plantillas HTML hermosas y responsivas.  
- [Creative Tim Free React Templates](https://www.creative-tim.com/templates/react-free) — Plantillas, kits y plantillas de tipo administrador gratuitas para React.  
- [Awesome Landing Page](https://github.com/nordicgiant2/awesome-landing-page) — Plantillas y ejemplos curados de páginas de aterrizaje.  

## Herramientas de diseño y edición de imágenes

Para maquetas, gráficos, capturas de pantalla, edición de imágenes ligera o diseño rápido basado en navegador.

- [Photopea](https://www.photopea.com/) — Edición similar a Photoshop directamente en el navegador.
- [Penpot](https://penpot.app/) — Diseño y prototipado de código abierto.
- [Corel Vector / Gravit Designer](https://designer.gravit.io/) — Diseño vectorial basado en navegador con una versión gratuita.
- [Boxy SVG](https://boxy-svg.com/) — Editor SVG enfocado para navegadores Chromium y WebKit.
- [VectorInk](https://vectorink.io/app/) — Dibujo y ilustración vectorial basados en navegador.
- [SVGOMG](https://jakearchibald.github.io/svgomg/) — Optimiza y limpia archivos SVG antes de su implementación.
- [Pixlr X](https://pixlr.com/x/) — Edición rápida en línea de imágenes cuando no necesitas una aplicación de diseño completa.

## Fuentes y herramientas de color

Las decisiones sobre tipografía y color tienen un impacto significativo en si una interfaz de usuario parece deliberada o genérica.

### Fuentes

- [Fontsource](https://fontsource.org) — Aloja fuentes de código abierto con instalaciones basadas en paquetes.
- [Google Fonts](https://fonts.google.com/) — El catálogo clásico de fuentes.
- [Fontpair](https://fontpair.co/fonts/) — Inspiración para combinaciones de fuentes cuando deseas tomar una decisión más rápida.
- [The Stocks](http://thestocks.im/) — Agrega fuentes, iconos, fotos y otros recursos creativos.

### Color y contraste

- [Adobe Color](https://color.adobe.com/create/color-wheel/) — Rueda de color clásica y herramientas de armonía.
- [Coolors](https://coolors.co/) — Genera, bloquea y ajusta paletas rápidamente.
- [Color Hunt](https://colorhunt.co/) — Inspiración para paletas curadas.
- [Paletton](https://paletton.com/) — Generador de esquemas de color con previsiones.
- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) — Comprueba las proporciones de contraste de WCAG.
- [Contrast Ratio](https://contrast-ratio.com/) — Pruebas rápidas de contraste con una interfaz más limpia.

## Animación e interacción

Usa estos cuando los diseños estáticos parezcan planos y necesites una mejor animación, estados de hover o navegación más dinámica.

- [GSAP](https://greensock.com/gsap) — El caballo de batalla de la industria para animaciones complejas de la interfaz de usuario.
- [Anime.js](https://animejs.com/) — Motor de animación JavaScript potente con una curva de aprendizaje más amigable.
- [Animate.css](https://animate.style/) — Animaciones predefinidas rápidas para entradas sencillas y retroalimentación.
- [Hover.css](https://ianlunn.github.io/Hover/) — Interacciones de hover y microefectos listos para usar.
- [Magic Animations](https://www.minimamente.com/project/magic/) — Animaciones CSS predefinidas.
- [Bounce.js](https://bouncejs.com/) — Generación de fotogramas clave similares a resortes. Archivado, pero aún útil para inspiración.
- [Hamburgers](https://github.com/jonsuh/hamburgers) — Animaciones de menú hamburguesa accesibles.
- [Best Snip SVG Animator](https://bestsnip.com/svg_animation/) — Herramienta web sencilla para animar SVG.
- [KokonutUI: Flow Field](https://kokonutui.com/docs/backgrounds/flow-field) — Receta de fondo animado con un fuerte impacto visual.
- [KokonutUI: Smooth Tab](https://kokonutui.com/docs/navigation/smooth-tab) — Patrón de navegación por pestañas suave que puedes adaptar para la interfaz de aplicación.

Estos son los recursos más fáciles de abusar. Prefiere una familia visual coherente en lugar de mezclar cinco estilos en una sola interfaz.

### Íconos

- [Lucide](https://lucide.dev/) — Íconos limpios y de código abierto, una excelente opción por defecto.
- [Iconoir](https://iconoir.com) — Conjunto grande de íconos SVG gratuitos con un aspecto ligeramente más suave.
- [Simple Icons](https://simpleicons.org/) — Íconos de marcas para productos y enlaces sociales.
- [Phosphor Icons](https://phosphoricons.com/) — Familia flexible de íconos para interfaces y aplicaciones.
- [IcoMoon App](https://icomoon.io/app/#/select) — Crea un conjunto personalizado de íconos y exporta paquetes SVG o PNG.
- [Font Awesome Free Search](https://fontawesome.com/v6/search?o=r&m=free) — Útil cuando necesitas variedad más que personalidad.
- [The Noun Project](https://thenounproject.com) — Biblioteca masiva de íconos con muchos objetos y conceptos novedosos.
- [SVG Repo](https://www.svgrepo.com/) — Biblioteca comunitaria SVG de gran tamaño. La calidad varía, así que cura con cuidado.

### Ilustraciones y personajes

- [Humaaans](https://www.humaaans.com/) — Ilustraciones modulares de personas para páginas de producto y marketing.
- [ManyPixels](https://www.manypixels.co/gallery/) — Biblioteca de ilustraciones actualizada con frecuencia.
- [DrawKit Free](https://www.drawkit.io/free) — Paquetes de ilustraciones gratuitas de alta calidad.
- [Open Doodles Generator](https://generator.opendoodles.com/) — Doodles generativos con estilo de dibujo a mano y aspecto juguetón.
- [Illlustrations](https://illlustrations.co/) — Estilo de ilustración a mano con un aire casual.
- [Icons8 Illustrations](https://icons8.com/illustrations) — Paquetes de ilustraciones gratuitas en múltiples estilos.
- [Isometric](https://isometric.online/) — Ilustraciones y activos isométricos.
- [React Kawaii](https://react-kawaii.vercel.app/#/Components) — Componentes SVG de personajes adorables.

## Licencias y consejos de uso

- No asumas que "gratis" significa "sin atribución". Revisa cada licencia antes de entregar trabajos para clientes.
- Ten cuidado con las restricciones de marcas comerciales al usar marcas, logotipos o iconos de aplicaciones en plantillas.
- Exporta SVG cuando sea posible para que los iconos y ilustraciones mantengan su nitidez y sean más fáciles de personalizar.
- Mantén coherentes las familias de iconos. Un icono limpio de Lucide al lado de un SVG de mercado cartoonesco suele verse accidental.
- Siempre prueba el contraste del texto. Muchas "paletas hermosas" se desmoronan al aplicarlas a textos largos.
- Trata las ilustraciones de stock como material bruto. Un poco de edición evita el efecto de plantilla copiada.
