## Start Here

Si solo necesitas unas cuantas opciones confiables, comienza con estas:

- Para bloques UI pulidos en React o Tailwind: [Aceternity UI](https://ui.aceternity.com/components), [Shadcnblocks](https://www.shadcnblocks.com/blocks) y [Tailark](https://tailark.com/blocks?licences=free).
- Para interfaces de productos de IA y patrones de chat: [Assistant UI](https://www.assistant-ui.com/) y [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought).
- Para íconos: [Lucide](https://lucide.dev/), [Iconoir](https://iconoir.com) y [Simple Icons](https://simpleicons.org/).
- Para ilustraciones: [ManyPixels](https://www.manypixels.co/gallery/), [DrawKit Free](https://www.drawkit.io/free) y [Humaaans](https://www.humaaans.com/).
- Para fuentes: [Fontsource](https://fontsource.org) y [Google Fonts](https://fonts.google.com/).
- Para color y accesibilidad: [Coolors](https://coolors.co/), [Adobe Color](https://color.adobe.com/create/color-wheel/) y [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/).

## UI Components, Blocks, and Templates

Esta es la sección que la mayoría de la gente realmente busca: plantillas de sitios web, secciones hero, shells de aplicaciones, bloques Tailwind y componentes UI que puedes adaptar en lugar de diseñar desde cero.

### AI and Chat UI

- [Assistant UI](https://www.assistant-ui.com/) — Componentes UI de chat IA de código abierto para aplicaciones React.  
- [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought) — Un patrón UI de chat estilo razonamiento limpio para productos de IA.  

### Bloques, Secciones y Componentes  

- [Aceternity UI](https://ui.aceternity.com/components) — Componentes React y Tailwind de alto acabado con fuerte personalidad visual.  
- [reui Collapsible](https://reui.io/components/collapsible) — Patrones UI limpios de colapsables y acordeón para interfaces React.  
- [Shadcnblocks](https://www.shadcnblocks.com/blocks) — Secciones y bloques listos para usar construidos sobre el ecosistema shadcn.  
- [Tailark](https://tailark.com/blocks?licences=free) — Bloques Tailwind gratuitos para sitios de marketing y páginas de producto.  
- [Tool UI](https://www.tool-ui.com/) — Bloques y componentes UI modernos para aplicaciones y páginas de aterrizaje.  
- [CSS Layout](https://csslayout.io/patterns) — Patrones de diseño listos para copiar y pegar para estructuras web comunes.  

### Plantillas completas e inspiración  

- [HTML5 UP](https://html5up.net) — Plantillas HTML hermosas y responsivas.  
- [Creative Tim Free React Templates](https://www.creative-tim.com/templates/react-free) — Dashboards, kits y plantillas de estilo admin gratuitos para React.  
- [Awesome Landing Page](https://github.com/nordicgiant2/awesome-landing-page) — Plantillas y ejemplos de páginas de aterrizaje curados.

## Fuentes y Herramientas de Color

La tipografía y las decisiones de color tienen un impacto desproporcionado en si una UI se siente deliberada o genérica.

### Fuentes

- [Fontsource](https://fontsource.org) — Aloja tus propias fuentes de código abierto mediante instalaciones basadas en paquetes.  
- [Google Fonts](https://fonts.google.com/) — El catálogo clásico de fuentes.  
- [Fontpair](https://fontpair.co/fonts/) — Inspiración para combinar fuentes cuando necesitas decidir más rápido.  
- [The Stocks](http://thestocks.im/) — Agrega fuentes, íconos, fotos y otros recursos creativos.

### Color and Contrast

- [Adobe Color](https://color.adobe.com/create/color-wheel/) — Herramientas clásicas de rueda de color y armonía.  
- [Coolors](https://coolors.co/) — Genera, bloquea y ajusta paletas rápidamente.  
- [Color Hunt](https://colorhunt.co/) — Inspiración de paletas curadas.  
- [Paletton](https://paletton.com/) — Generador de esquemas de color con vistas previas.  
- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) — Verifica ratios de contraste WCAG.  
- [Contrast Ratio](https://contrast-ratio.com/) — Pruebas rápidas de contraste con una interfaz más limpia.

- [Anime.js](https://animejs.com/) — Motor de animación JavaScript potente con una curva de aprendizaje más amigable.  
- [Animate.css](https://animate.style/) — Animaciones predefinidas rápidas para entradas simples y retroalimentación.  
- [Hover.css](https://ianlunn.github.io/Hover/) — Interacciones hover y micro‑efectos listos para usar.  
- [Magic Animations](https://www.minimamente.com/project/magic/) — Animaciones CSS predefinidas.  
- [Bounce.js](https://bouncejs.com/) — Generador de fotogramas clave tipo resorte. Archivado, pero útil para inspiración.  
- [Hamburgers](https://github.com/jonsuh/hamburgers) — Animaciones de menú hamburguesa accesibles.  
- [Best Snip SVG Animator](https://bestsnip.com/svg_animation/) — Herramienta web sencilla para animar SVGs.  
- [KokonutUI: Flow Field](https://kokonutui.com/docs/backgrounds/flow-field) — Receta de fondo animado con un fuerte impacto visual.  
- [KokonutUI: Smooth Tab](https://kokonutui.com/docs/navigation/smooth-tab) — Patrón de navegación por pestañas suave que puedes adaptar a UI de apps.

- [Lucide](https://lucide.dev/) — Íconos limpios, de código abierto y una excelente opción predeterminada.  
- [Iconoir](https://iconoir.com) — Gran conjunto de íconos SVG gratuitos con un aspecto ligeramente más suave.  
- [Simple Icons](https://simpleicons.org/) — Íconos de marcas para productos y enlaces sociales.  
- [Phosphor Icons](https://phosphoricons.com/) — Familia de íconos flexible para interfaces y aplicaciones.  
- [IcoMoon App](https://icomoon.io/app/#/select) — Crea un conjunto de íconos personalizado y exporta paquetes SVG o PNG.  
- [Font Awesome Free Search](https://fontawesome.com/v6/search?o=r&m=free) — Útil cuando necesitas amplitud más que personalidad.  
- [The Noun Project](https://thenounproject.com) — Biblioteca masiva de íconos con abundantes objetos y conceptos de nicho.

## Licencias y consejos de uso

- No asumas que “gratis” implica “sin atribución”. Revisa cada licencia antes de entregar trabajo al cliente.
- Vigila las restricciones de marcas registradas al usar logotipos, marcas o íconos de aplicaciones en plantillas.
- Exporta SVG siempre que sea posible para que los íconos e ilustraciones mantengan su nitidez y sea más fácil tematizarlos.
- Mantén la coherencia entre familias de íconos. Un ícono Lucide limpio junto a un SVG caricaturesco de marketplace suele verse como un accidente.
- Siempre prueba el contraste del texto. Muchas paletas “hermosas” se desmoronan con texto real.
- Trata las ilustraciones de stock como materia prima. Un poco de edición ayuda mucho a evitar esa sensación de plantilla copiada.
