## はじめに

数点だけ信頼できる選択肢が欲しいなら、まずここから始めてください。

- 洗練された React または Tailwind の UI ブロック: [Aceternity UI](https://ui.aceternity.com/components)、[Shadcnblocks](https://www.shadcnblocks.com/blocks)、[Tailark](https://tailark.com/blocks?licences=free)。
- AI 製品のインターフェースやチャットパターン: [Assistant UI](https://www.assistant-ui.com/) と [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought)。
- アイコン: [Lucide](https://lucide.dev/)、[Iconoir](https://iconoir.com)、[Simple Icons](https://simpleicons.org/)。
- イラスト: [ManyPixels](https://www.manypixels.co/gallery/)、[DrawKit Free](https://www.drawkit.io/free)、[Humaaans](https://www.humaaans.com/)。
- フォント: [Fontsource](https://fontsource.org) と [Google Fonts](https://fonts.google.com/)。
- カラーとアクセシビリティ: [Coolors](https://coolors.co/)、[Adobe Color](https://color.adobe.com/create/color-wheel/)、[WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/)。

## UI コンポーネント、ブロック、テンプレート

ほとんどの人が実際に求めているのはここです。ウェブサイトテンプレート、ヒーローセクション、アプリシェル、Tailwind ブロック、そしてゼロからデザインする代わりに適用できる UI コンポーネントです。

### AI とチャット UI

- [Assistant UI](https://www.assistant-ui.com/) — React アプリ向けのオープンソース AI チャット UI コンポーネント。
- [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought) — AI 製品向けのクリーンな推論スタイルチャット UI パターン。

### ブロック、セクション、コンポーネント

- [Aceternity UI](https://ui.aceternity.com/components) — 高い完成度と強いビジュアル個性を持つ React と Tailwind のコンポーネント。
- [reui Collapsible](https://reui.io/components/collapsible) — React インターフェース向けの洗練された折りたたみ・アコーディオン UI パターン。
- [Shadcnblocks](https://www.shadcnblocks.com/blocks) — shadcn エコシステムを基盤にした即利用可能なセクションとブロック。
- [Tailark](https://tailark.com/blocks?licences=free) — マーケティングサイトや製品ページ向けの無料 Tailwind ブロック。
- [Tool UI](https://www.tool-ui.com/) — アプリやランディングページ向けのモダン UI ブロックとコンポーネント。
- [CSS Layout](https://csslayout.io/patterns) — 一般的なウェブサイト構造用のコピペ可能なレイアウトパターン。

### 完全テンプレートとインスピレーション

- [HTML5 UP](https://html5up.net) — 美しくレスポンシブな HTML テンプレート。
- [Creative Tim Free React Templates](https://www.creative-tim.com/templates/react-free) — 無料の React ダッシュボード、キット、管理画面スタイルテンプレート。
- [Awesome Landing Page](https://github.com/nordicgiant2/awesome-landing-page) — キュレーションされたランディングページテンプレートと例。

## デザインと画像編集ツール

モックアップ、グラフィック、スクリーンショット、軽量な画像編集、または高速なブラウザベースのデザイン作業に最適です。

- [Photopea](https://www.photopea.com/) — ブラウザ上で Photoshop に似た編集が可能。
- [Penpot](https://penpot.app/) — オープンソースのデザイン・プロトタイピングツール。
- [Corel Vector / Gravit Designer](https://designer.gravit.io/) — 無料プランで利用できるブラウザベースのベクターデザイン。
- [Boxy SVG](https://boxy-svg.com/) — Chromium と WebKit ブラウザ向けに特化した SVG エディタ。
- [VectorInk](https://vectorink.io/app/) — ブラウザ上でベクタードローイングとイラスト作成ができるツール。
- [SVGOMG](https://jakearchibald.github.io/svgomg/) — 配布前に SVG ファイルを最適化・クリーンアップ。
- [Pixlr X](https://pixlr.com/x/) — フル機能のデザインアプリが不要なときの、手軽なオンライン画像編集。

## フォントとカラー ツール

タイポグラフィとカラーの選択は、UI が意図的に見えるか汎用的に見えるかを大きく左右します。

### フォント

- [Fontsource](https://fontsource.org) — パッケージベースでインストールできる、セルフホスト型オープンソースフォント。
- [Google Fonts](https://fonts.google.com/) — 定番のフォントカタログ。
- [Fontpair](https://fontpair.co/fonts/) — 素早く決定したいときのフォント組み合わせインスピレーション。
- [The Stocks](http://thestocks.im/) — フォント、アイコン、写真、その他クリエイティブリソースをまとめて提供。

### カラーとコントラスト

- [Adobe Color](https://color.adobe.com/create/color-wheel/) — 従来のカラーホイールと調和ツール。
- [Coolors](https://coolors.co/) — パレットをすばやく生成、ロック、調整。
- [Color Hunt](https://colorhunt.co/) — キュレーションされたパレットのインスピレーション。
- [Paletton](https://paletton.com/) — プレビュー付きの配色スキームジェネレータ。
- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) — WCAG コントラスト比をチェック。
- [Contrast Ratio](https://contrast-ratio.com/) — クリーンなインターフェースで高速コントラストテスト。

## アニメーションとインタラクション

- [GSAP](https://greensock.com/gsap) — 複雑な UI アニメーションの業界標準ツール。
- [Anime.js](https://animejs.com/) — 学習コストが低めの強力な JavaScript アニメーションエンジン。
- [Animate.css](https://animate.style/) — シンプルな出入りやフィードバック向けの高速プリセットアニメーション。
- [Hover.css](https://ianlunn.github.io/Hover/) — 用意されたホバーインタラクションとマイクロエフェクト。
- [Magic Animations](https://www.minimamente.com/project/magic/) — プリセット CSS アニメーション。
- [Bounce.js](https://bouncejs.com/) — スプリング感覚のキーフレーム生成。アーカイブ済みだがインスピレーションには有用。
- [Hamburgers](https://github.com/jonsuh/hamburgers) — アクセシブルなハンバーガーメニューアニメーション。
- [Best Snip SVG Animator](https://bestsnip.com/svg_animation/) — SVG をアニメーション化するシンプルなウェブツール。
- [KokonutUI: Flow Field](https://kokonutui.com/docs/backgrounds/flow-field) — 強いビジュアルインパクトを持つアニメーション背景レシピ。
- [KokonutUI: Smooth Tab](https://kokonutui.com/docs/navigation/smooth-tab) — アプリ UI に適用できるスムーズタブナビゲーションパターン。

## アイコンとイラスト

これらは過度に使用すると見た目が崩れやすいリソースです。5 つの異なるスタイルを同一インターフェイスに混ぜるよりも、一貫したビジュアルファミリーを選択してください。

### アイコン

- [Lucide](https://lucide.dev/) — クリーンでオープンソースのアイコン。デフォルトの選択肢として優秀です。
- [Iconoir](https://iconoir.com) — やや柔らかい印象の大規模な無料 SVG アイコンセット。
- [Simple Icons](https://simpleicons.org/) — 製品やソーシャルリンク向けのブランドアイコン。
- [Phosphor Icons](https://phosphoricons.com/) — インターフェイスやアプリ向けの柔軟なアイコンファミリー。
- [IcoMoon App](https://icomoon.io/app/#/select) — カスタムアイコンセットを作成し、SVG または PNG バンドルとしてエクスポートできます。
- [Font Awesome Free Search](https://fontawesome.com/v6/search?o=r&m=free) — 個性よりも網羅性が必要なときに便利です。
- [The Noun Project](https://thenounproject.com) — ニッチなオブジェクトや概念を多数含む巨大なアイコンライブラリ。
- [SVG Repo](https://www.svgrepo.com/) — 膨大なコミュニティ提供の SVG ライブラリ。品質にばらつきがあるため、慎重に選別してください。

### イラストとキャラクター

- [Humaaans](https://www.humaaans.com/) — プロダクトやマーケティングページ向けのモジュラー人物イラスト。
- [ManyPixels](https://www.manypixels.co/gallery/) — 定期的に更新されるイラストライブラリ。
- [DrawKit Free](https://www.drawkit.io/free) — 高品質な無料イラストパック。
- [Open Doodles Generator](https://generator.opendoodles.com/) — 手描き感のある遊び心あるドゥードルを生成。
- [Illlustrations](https://illlustrations.co/) — カジュアルな雰囲気の手描きイラストスタイル。
- [Icons8 Illustrations](https://icons8.com/illustrations) — 複数のスタイルにわたる無料イラストパック。
- [Isometric](https://isometric.online/) — 等角投影のイラストとアセット。
- [React Kawaii](https://react-kawaii.vercel.app/#/Components) — かわいい SVG キャラクターコンポーネント。

## ライセンスと利用上の注意

- 「無料」＝「クレジット不要」と決めつけないでください。クライアント案件に組み込む前に各ライセンスを必ず確認しましょう。
- テンプレートでブランドマーク、ロゴ、アプリアイコンを使用する場合、商標制限に注意してください。
- 可能な限り SVG でエクスポートし、アイコンやイラストをシャープに保ち、テーマ変更を容易にします。
- アイコンファミリーは統一してください。クリーンな Lucide アイコンの横に、漫画調のマーケットプレイス SVG が混在すると、意図しない印象になります。
- テキストのコントラストは必ずテストしましょう。多くの「美しい」パレットは実際の本文では崩れやすいです。
- ストックイラストは素材として扱い、少し手を加えるだけでテンプレート感を大幅に軽減できます。
