## UI Components, Blocks, and Templates

実際に多くの人が求めているのはこのセクションです。ウェブサイトテンプレート、ヒーローセクション、アプリシェル、Tailwind ブロック、そしてゼロからデザインする代わりにそのまま流用できる UI コンポーネントが揃っています。

### AI and Chat UI

- [Assistant UI](https://www.assistant-ui.com/) — React アプリ向けのオープンソース AI チャット UI コンポーネント。
- [AI SDK Elements: Chain of Thought](https://elements.ai-sdk.dev/components/chain-of-thought) — AI 製品向けのクリーンな推論スタイルチャット UI パターン。

### ブロック、セクション、コンポーネント

- [Aceternity UI](https://ui.aceternity.com/components) — ビジュアルに個性があり、完成度の高い React と Tailwind コンポーネント。
- [reui Collapsible](https://reui.io/components/collapsible) — React インターフェース向けのシンプルな折りたたみ・アコーディオン UI パターン。
- [Shadcnblocks](https://www.shadcnblocks.com/blocks) — shadcn エコシステムを基盤にした、すぐ使えるセクションとブロック。
- [Tailark](https://tailark.com/blocks?licences=free) — マーケティングサイトやプロダクトページ向けの無料 Tailwind ブロック。
- [Tool UI](https://www.tool-ui.com/) — アプリやランディングページ向けのモダン UI ブロックとコンポーネント。
- [CSS Layout](https://csslayout.io/patterns) — 一般的なウェブサイト構造のコピペ可能なレイアウトパターン。

### 完全テンプレートとインスピレーション

- [HTML5 UP](https://html5up.net) — 美しくレスポンシブな HTML テンプレート。
- [Creative Tim Free React Templates](https://www.creative-tim.com/templates/react-free) — 無料の React ダッシュボード、キット、管理画面テンプレート。
- [Awesome Landing Page](https://github.com/nordicgiant2/awesome-landing-page) — キュレーションされたランディングページテンプレートと例。

- [Simple Icons](https://simpleicons.org/) — 製品やソーシャルリンク向けのブランドアイコン。
- [Phosphor Icons](https://phosphoricons.com/) — インターフェイスやアプリ向けに柔軟に使えるアイコンファミリー。

- [Icons8 Illustrations](https://icons8.com/illustrations) — 複数のスタイルにわたる無料イラストパック。

## ライセンスと利用上の注意

- 「無料」だからといって「クレジット不要」と決めつけないでください。クライアント案件に組み込む前に各ライセンスを必ず確認しましょう。

- ストックイラストは素材として扱い、少し手を加えるだけでテンプレート感を大幅に低減できます。
