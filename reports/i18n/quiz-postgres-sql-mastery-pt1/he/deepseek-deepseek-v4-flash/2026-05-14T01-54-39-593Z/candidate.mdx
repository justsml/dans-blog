  <div className="explanation">

    ```sql
        CREATE TABLE student_emails (
          student_id INTEGER,

          UNIQUE(email)
        );
    ```

    **`timestamptz`**
    - מנרמל את הקלט לנקודת זמן מוחלטת.

    - מאחסן את התאריך והשעה ללא מידע על אזור זמן.
    - שימושי לאחסון מפורש של תאריכים מתוקננים, בין אם ב-UTC או באזור זמן ספציפי.

