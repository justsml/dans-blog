  ]}
>
  <slot name="question">
  <div className="question">
    SQLにおける`WITH`句（Common Table Expression）の目的は何ですか？
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    `WITH`句（Common Table Expression、CTE）は、メインクエリ内で参照可能な一時的な結果セットを定義するために使用されます。PostgreSQLでは、`WITH`は`SELECT`、`INSERT`、`UPDATE`、`DELETE`、`MERGE`などの文に付加できます。

    これにより、複雑なクエリの可読性と保守性が向上します。

    構文は次の通りです：
    ```sql
    WITH cte_name AS (
      SELECT column_name
      FROM table_name
    )
    SELECT *
    FROM cte_name;
    ```
  </div>
  </slot>
</Challenge>
