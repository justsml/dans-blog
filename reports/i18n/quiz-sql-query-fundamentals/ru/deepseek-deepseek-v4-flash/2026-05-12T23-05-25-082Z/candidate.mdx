    Какова цель предложения `WITH` (общее табличное выражение) в SQL?
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    Предложение `WITH`, или общее табличное выражение (CTE), используется для определения временного набора результатов, на который можно ссылаться в основном запросе. В PostgreSQL `WITH` может быть присоединено к таким операторам, как `SELECT`, `INSERT`, `UPDATE`, `DELETE` или `MERGE`.

    Это помогает улучшить читаемость и поддерживаемость сложных запросов.

    Синтаксис:
    ```sql
    WITH cte_name AS (
      SELECT column_name
      FROM table_name
    )
    SELECT *
    FROM cte_name;
    ```
  </div>
