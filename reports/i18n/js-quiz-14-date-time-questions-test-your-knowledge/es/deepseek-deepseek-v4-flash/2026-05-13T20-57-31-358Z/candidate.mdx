## ¿Qué tan bien conoces la clase `Date`?

> * **¡Demuestra tus habilidades en JavaScript!** 🚀
> * No se requiere inicio de sesión ni registro. ✨
> * Opción múltiple. 🤖 ... _¿Qué tan difícil puede ser, eh?_

### Esquema

La clase `Date` en JavaScript tiene una API notoriamente difícil. Fue heredada de Java, y solo puedo suponer que se inspiró en métodos antiguos neolíticos de medición del tiempo.

La lucha por entender `Date` lleva a muchos desarrolladores a usar bibliotecas de terceros sin cuestionarlo. Si bien a menudo es una opción segura y confiable, estas bibliotecas rara vez son necesarias para formatear fechas o localización.

Este cuestionario está diseñado para probar (y profundizar) tu conocimiento de la API nativa `Date`. ¡Usa los botones verdes para pistas y explicaciones! Con suerte, al final del desafío habrás consolidado tu comprensión de `Date` en JavaScript.

#### **NOTA:** Asume que todos los ejemplos asumen la zona horaria local GMT-7.

    La forma `T00:00` hace que el segundo valor se interprete como medianoche local.

    La primera fecha se interpreta como `Tue Dec 31 2019 17:00:00 GMT-0700 (Mountain Standard Time)`.
