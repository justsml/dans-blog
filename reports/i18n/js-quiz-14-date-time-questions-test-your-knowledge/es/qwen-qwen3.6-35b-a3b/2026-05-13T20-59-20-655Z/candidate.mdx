## ¿Qué tan bien conoces la clase `Date`?

> * **¡Demuestra tus habilidades en JavaScript!** 🚀
> * No requiere inicio de sesión ni registro. ✨
> * Opción múltiple. 🤖 ... _¿Qué tan difícil puede ser, eh?_

### Contenido

La clase `Date` en JavaScript tiene una API notoriamente difícil. Fue heredada de Java, y solo puedo asumir que se inspiró en métodos neolíticos de medición del tiempo.

La dificultad para manejar `Date` lleva a muchos desarrolladores a usar bibliotecas de terceros sin cuestionarlo. Aunque suelen ser una opción segura y confiable, ¡rara vez se necesitan para dar formato a fechas o localización!

Este cuestionario está diseñado para evaluar (y profundizar) tu conocimiento de la API nativa `Date`. ¡Usa los botones verdes para obtener pistas y explicaciones! Al finalizar el desafío, espero que hayas consolidado tu comprensión del manejo de `Date` en JavaScript.

#### **NOTA:** Asume que todos los ejemplos asumen la zona horaria local GMT-7.

    Las cadenas de fecha y hora sin una zona horaria explícita se interpretan en hora local.

    El formato `T00:00` hace que el segundo valor se interprete como la medianoche local.

    La primera fecha se interpreta como `Tue Dec 31 2019 17:00:00 GMT-0700 (Mountain Standard Time)`.
    La segunda fecha se interpreta como `Wed Jan 01 2020 00:00:00 GMT-0700 (Mountain Standard Time)`.
