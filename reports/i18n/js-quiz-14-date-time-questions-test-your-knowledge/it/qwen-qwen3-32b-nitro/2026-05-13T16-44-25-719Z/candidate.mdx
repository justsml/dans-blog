## Quanto conosci la classe `Date`?

> * **Dimostra le tue skill in JavaScript!** 🚀
> * Nessun login o iscrizione richiesti. ✨
> * Domande a risposta multipla. 🤖 ... _Quanto può essere difficile, eh?_

### Schema

La classe `Date` in JavaScript ha un API notoriamente difficile. È stata ereditata da Java e posso solo immaginare che fosse ispirata a metodi antichi di conservazione del tempo.

La lotta per ottenere `Date`'s porta molti sviluppatori a utilizzare librerie di terze parti senza esitazione. Sebbene spesso siano una scelta sicura e affidabile, raramente sono necessarie per la formattazione delle date o la localizzazione!

Questo quiz è progettato per testare (e approfondire) le tue conoscenze sull'API nativa `Date`. Usa i pulsanti verdi per suggerimenti e spiegazioni! Spero che alla fine della sfida avrai consolidato la tua comprensione delle `Date` in JavaScript.

#### **NOTA:** Si assume che tutti gli esempi siano nel fuso orario GMT-7.

    Le stringhe data-ora senza fuso orario esplicito vengono interpretate in orario locale.

    La forma `T00:00` fa sì che il secondo valore venga interpretato come mezzanotte locale.

    La prima data viene interpretata come `Tue Dec 31 2019 17:00:00 GMT-0700 (Mountain Standard Time)`.
    La seconda data viene interpretata come `Wed Jan 01 2020 00:00:00 GMT-0700 (Mountain Standard Time)`.

    <aside class="hint">
