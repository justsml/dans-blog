## `Date` क्लास को आप कितनी अच्छी तरह जानते हैं?

> * **अपनी JavaScript कौशल साबित करें!** 🚀  
> * कोई लॉगिन या साइन‑अप नहीं चाहिए। ✨  
> * मल्टीपल चॉइस। 🤖 … _कितना मुश्किल हो सकता है, है ना?_

### रूपरेखा

JavaScript में `Date` क्लास का API कुख्यात रूप से जटिल है। यह Java से विरासत में मिला है, और संभवतः प्राचीन नियोलिथिक समय‑मापन तरीकों से प्रेरित था।

`Date` को समझने की कठिनाई कई डेवलपर्स को बिना सवाल पूछे थर्ड‑पार्टी लाइब्रेरीज़ इस्तेमाल करने पर मजबूर करती है। अक्सर ये सुरक्षित और भरोसेमंद विकल्प होते हैं, लेकिन तारीख फॉर्मेटिंग या लोकलाइज़ेशन के लिए इन्हें वास्तव में ज़रूरत नहीं होती!

यह क्विज़ आपके नेटीव `Date` API के ज्ञान को परखने (और गहरा करने) के लिए बनाया गया है। संकेत और व्याख्याओं के लिए हरे बटन का उपयोग करें! आशा है कि चुनौती के अंत तक आप JavaScript में `Date` की समझ को पक्का कर लेंगे।

#### **नोट:** सभी उदाहरण GMT‑7 स्थानीय टाइमज़ोन मानते हैं।
