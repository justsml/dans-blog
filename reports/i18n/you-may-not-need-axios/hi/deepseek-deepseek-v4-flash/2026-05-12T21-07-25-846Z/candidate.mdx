### सुविधा तुलना

|                                                 | fetch    | axios    | request |
|-------------------------------------------------|:--------:|:--------:|:-------:|

| अनुरोध और प्रतिक्रिया डेटा को रूपांतरित करना             |✅        |✅         |✅       |
| अनुरोध रद्द करना                                 |✅        |✅         |❌       |

<br /><br />

जब मैंने यह लेख लिखना शुरू किया (देर 2018, 2024 में अपडेट किया गया), तो मुझे लगा था कि मैं मिश्रित चेकबॉक्स वाली एक तालिका के साथ समाप्त करूँगा। निश्चित रूप से कुछ विशेष _उपयोग के मामले_ हैं जो [`axios`](https://www.npmjs.com/package/axios), [`request`](https://www.npmjs.com/package/request), [`r2`](https://www.npmjs.com/package/r2), [`superagent`](https://www.npmjs.com/package/superagent), [`got`](https://www.npmjs.com/package/got) आदि को उचित ठहराते हैं।

खैर, जैसा कि पता चला, **मैंने तृतीय-पक्ष HTTP लाइब्रेरियों की आवश्यकता को अधिक आंका था।

**

कई वर्षों तक `fetch` का उपयोग करने के बावजूद (गैर-सामान्य कार्यों सहित: फ़ाइल अपलोड और त्रुटि/पुनर्प्रयास समर्थन), मुझे अभी भी `fetch` की क्षमताओं और सीमाओं के बारे में गलत धारणाएँ थीं।

नेटिव `fetch` स्वचालित रूप से JSON प्रतिक्रियाओं को पार्स नहीं करता है या JSON अनुरोध निकायों को स्ट्रिंगिफाई नहीं करता है।

आप वापसी पर `response.json()` और भेजते समय `JSON.stringify()` को कॉल करते हैं। एर्गोनॉमिक्स के उस मामले में Axios अभी भी बेहतर है; `fetch` के पक्ष में तर्क यह है कि एक छोटी सहायक फ़ंक्शन अक्सर इस अंतर को भर देती है।

खैर, आइए देखते हैं कि `fetch` क्या कर सकता है...

### संगतता

2022 तक, `fetch` API सभी आधुनिक ब्राउज़रों और NodeJS v18+ के नए संस्करणों में [व्यापक रूप से समर्थित](https://caniuse.com/#feat=fetch) है।

यदि आपको IE को सपोर्ट करना ही है, तो आप `github/fetch` पैकेज (GitHub की एक शानदार टीम द्वारा अनुरक्षित) के साथ [fetch को पॉलीफिल](https://github.com/github/fetch#browser-support) कर सकते हैं। [IE8](https://github.com/camsong/fetch-ie8) तक जाना संभव है - _आपकी माइलेज भिन्न हो सकती है_।

पुराने NodeJS में [`node-fetch`](https://www.npmjs.com/package/node-fetch) पैकेज के साथ `fetch` API का लाभ उठाया जा सकता है:

```sh
npm install node-fetch
```

_पॉलीफिल+node-fetch के बाद: 99.99% संगत_ ✅
