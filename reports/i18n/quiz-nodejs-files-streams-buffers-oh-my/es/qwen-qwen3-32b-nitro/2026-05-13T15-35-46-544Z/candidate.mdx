        const upperCase = new Transform({

          }
        });

    ```

  <slot name='explanation'>
  <div className="explanation">
    Los flujos de transformación modifican los datos al pasar. Aquí, cada fragmento se:
    1. Convierte a cadena
    2. Transforma a mayúsculas
    3. Pasa a stdout

    Esto crea una tubería que convierte toda la entrada en mayúsculas.

    [Aprende más sobre flujos de transformación](https://nodejs.org/api/stream.html#class-streamtransform)

    ```

    2. Lo renombran al archivo objetivo

    - Usar el paquete `chokidar`
    - Usar un debounce en el callback
    - Usar `fs.watchFile()` (aunque es menos eficiente)

  </div>

  <slot name='explanation'>
  <div className="explanation">
    La presión de retroalimentación es un mecanismo que evita el desbordamiento de memoria pausando la lectura cuando el extremo de escritura no puede mantenerse al día.

    Ejemplo de presión de retroalimentación manual:
    ```js
        readable.on('data', (chunk) => {
          const canContinue = writable.write(chunk);
          if (!canContinue) {
            readable.pause();
            writable.once('drain', () => readable.resume());
          }
        });
    ```
    ¡pipe() maneja esto automáticamente!

    [Aprende más sobre la presión de retroalimentación](https://nodejs.org/en/learn/modules/backpressuring-in-streams#backpressuring-in-streams)
  </div>
  </slot>
