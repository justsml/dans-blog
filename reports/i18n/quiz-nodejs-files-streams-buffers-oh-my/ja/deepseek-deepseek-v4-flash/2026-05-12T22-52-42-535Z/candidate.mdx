    - `chokidar` パッケージ
    - コールバックのデバウンス
    - `fs.watchFile()` の使用（ただし効率は低い）

    手動バックプレッシャーの例：
    ```js
        readable.on('data', (chunk) => {
          const canContinue = writable.write(chunk);
          if (!canContinue) {
            readable.pause();
            writable.once('drain', () => readable.resume());
          }
        });
    ```
    `pipe()` はこれを自動的に処理します！

    [バックプレッシャーの詳細](https://nodejs.org/en/learn/modules/backpressuring-in-streams#backpressuring-in-streams)
