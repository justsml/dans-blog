    Например, `#ABC` эквивалентно `#AABBCC`, а `#ABCD` эквивалентно `#AABBCCDD`. Чтобы узнать больше о работе с hex-значениями, посмотрите мою [викторину по числам JavaScript.](/quiz-can-you-count-to-bigint/)
