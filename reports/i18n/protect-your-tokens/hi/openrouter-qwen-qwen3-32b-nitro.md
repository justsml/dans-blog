## 🔒 `गुप्त कुंजियाँ`

**‼️ महत्वपूर्ण:** `गुप्त कुंजियाँ` **अवश्य** Git द्वारा नजरअंदाज किए जाने चाहिए _और_ सभी ब्राउज़र कोड में छोड़ दिए जाने चाहिए। [dotenv का उपयोग कैसे करें](#-how-to-handle-secrets-safely)  

<br />  

_आप कैसे जानें कि आप एक `गुप्त कुंजी` के साथ काम कर रहे हैं?_  

<br />  

**👍 अंतर्ज्ञान:** `CORS त्रुटियाँ` वापस करने वाले सर्वर ब्राउज़र समर्थन की कमी का संकेत देते हैं। यह बलपूर्वक संकेत देता है कि आपको सेवा को प्रॉक्सी करना होगा, जैसे कि यह `गुप्त` हो।  

**👍 अंतर्ज्ञान:** लागत वाली सेवाएँ (लगभग) हमेशा प्रॉक्सी कर देनी चाहिए या छिपानी चाहिए।

**👍 अंतर्ज्ञान:** यदि आप एक लिखने का संचालन (**फ़ाइल अपलोड, डेटाबेस पंक्ति डालें**) करते हैं, तो आप `गुप्त कुंजी` के साथ काम कर रहे हो सकते हैं।  

<br />  

**_उपयोग के मामले और सुविधाएँ:_** `गुप्त` कुंजी  

- लंबी अवधि की अधिकृतता (प्रमाणक, पहुंच टोकन, JSON वेब टोकन)  
- छोटी अवधि की अधिकृतता (OAuth टोकन, सत्र संग्रह)  
- भुगतान योग्य/महंगी सेवाओं का उपयोग (प्रमाणीकरण, भौगोलिक कोडिंग, फ़ाइल संग्रहण, आदि)  
- सार्वजनिक/निजी जोड़े का निजी भाग (RECAPTCHA, Stripe, Auth0)  
- सेवा प्रमाणक (ईमेल/एसएमटीपी, एलडीएपी/निर्देशिका सेवाएँ)  
- डेटा एन्क्रिप्शन और पूर्णता जाँच  

### चेकलिस्ट: गुप्त चाबियों का सुरक्षित रूप से निपटाना  

#### त्वरित सर्वेक्षण

अपने कोड में **गुप्त चाबियों को खत्म करने** के लिए निम्नलिखित चरणों को पूरा करें:  

- [ ] प्रत्यक्ष लिखित चाबियों को पर्यावरण चर से बदलें। उदाहरण: `process.env.API_SECRET`  
- [ ] एक पुस्तकालय का उपयोग करें, जैसे [`dotenv`](https://github.com/motdotla/dotenv#dotenv), और एक `.env` फ़ाइल के साथ। अपनी पहले से प्रत्यक्ष लिखित गुप्त चाबियों को `.env` फ़ाइल में जोड़ें।  
- [ ] अपने `.gitignore` फ़ाइल में एक `.env` पंक्ति जोड़ें!  

> **नहीं करें** डिप्लॉय किए गए सर्वरों पर एक `.env` फ़ाइल बनाएं। अपनी होस्टिंग सेवाओं के (जैसे [Heroku](https://devcenter.heroku.com/articles/config-vars), Netlify, AWS EC2) द्वारा प्रदान किए गए पर्यावरण चर प्रबंधन उपकरण का उपयोग करें: उदाहरण के लिए **डैशबोर्ड या कमांड लाइन**।

## 🌍 `गैर-गुप्त चाबियाँ`

**👍 अंगूठा का नियम:** जब कोई चाबी कोड या इनलाइन में ब्राउज़र में भेजी जानी है (उदाहरण के लिए `<script src="https://my-api/?apiKey=123-abc-456">` टैग के माध्यम से), **तो यह निश्चित रूप से एक `गैर-गुप्त` है**। एक सामान्य उदाहरण Google Maps है।  

<br />  

**_उपयोग के मामले और विशेषताएँ:_** `गैर-गुप्त` चाबियाँ  

- अल्पकालिक एक्सेस (यूजर सेशन IDs, JSON Web Tokens)  
- एप्प/विकासकर्ता द्वारा API एक्सेस की सीमा (ऑथ, जियोकोडिंग आदि)  
- सार्वजनिक/निजी जोड़े का सार्वजनिक भाग (RECAPTCHA, Stripe, Auth0)  
- एनालिटिक्स IDs  

#### ✅ `गैर-गुप्त` का संचालन:  

> **`गैर-गुप्त` (सार्वजनिक) चाबियों को हार्ड-कोड करना सुरक्षित है!**

अपने एप्प के लंबे समय तक प्रबंधन को आसान बनाएं एक साझा `config.js` के साथ।  

**उदाहरण:**  

```js
// config.js
module.exports = {
  googleMapsKey: '123-abc'
};
```

```js
// load-map.js
const config = require('./config.js');
const key = config.googleMapsKey;
const src = `//maps.googleapis.com/maps/api/js?key=${key}`;
// ...
```
