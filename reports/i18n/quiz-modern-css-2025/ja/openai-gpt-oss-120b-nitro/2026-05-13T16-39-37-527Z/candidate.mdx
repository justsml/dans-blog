>
  <slot name="question">
  <div className="question">
    親/コンテナの幅が 400px の場合、以下の要素の計算後の幅はどうなりますか？
    ```css
        div {
          width: min(250px, 50%);
        }
    ```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    `min()` 関数は、250px と親の幅の 50% のうち小さい方の値を選びます。

    計算された値を理解するには、相対単位をピクセルに変換する必要があります:

    - `400px` の `50%` は `200px`
    - `250px` はすでにピクセルです
    ```css
        /* This gets computed to */
        width: min(250px, 200px);
        /* -> 200px wins */
    ```
    `min()` 関数はレスポンシブデザインで特に便利で、コンポーネント（やフォントサイズ）がある上限を超えないように保証できます。
  </div>
  </slot>
</Challenge>

  <slot name='explanation'>
  <div className="explanation">
    `max()` 関数は 2 つ以上の入力を受け取り、自動的に最大の値を使用します。ルートフォントサイズがブラウザ既定の `16px` と仮定すると、幅は `96px` になります。

    計算された値を理解するには、相対単位をピクセルに変換する必要があります:

    - `50px` はすでにピクセルです
    - `200px` の `10%` は `20px` です
    - `6rem` は `6 * 16px`（既定のフォントサイズ）で、`96px` です
    ```css
        /* This gets computed to */
        width: max(50px, 20px, 96px);
        /* -> 96px wins */
    ```
  </div>

  options={[

    {text: '`vw` 単位が 20px から 50px の間になるように保証'},

    {text: '失敗！IE 11 はサポートされていません'},
  ]}
>
  <slot name="question">
  <div className="question">
    `clamp()` は何をするものですか？
    ```css
        .card {
          width: clamp(200px, 50vw, 500px);
        }
    ```
  </div>
  </slot>
  <slot name='explanation'>
  <div className="explanation">
    `clamp()` 関数は幅を `50vw` に基づいてスケールさせつつ、200px から 500px の範囲に収めます。

    つまり、`50vw` が 200px 未満になる場合は幅が 200px、`50vw` が 500px を超える場合は幅が 500px となり、その間は線形に変化します。

    これで自動的にレスポンシブになります！ `clamp` のポイントは **固定単位** と **レスポンシブまたは計算単位** を組み合わせることです。

    通常、フォントサイズにビューポート単位を使うのは避けるべきですが、`clamp()` を使えばフォントサイズが小さすぎたり大きすぎたりするのを防げます。
  </div>
  </slot>
</Challenge>
